🚗 CarDekho Analytics Dashboard

An interactive data analytics dashboard built using Python, Streamlit, Pandas, and Plotly to analyze CarDekho car dataset data. The project provides visual insights into car prices, fuel types, transmission, mileage, car age, and other important trends through dynamic charts and filters.

📌 Features
📊 Interactive Dashboard UI
🔍 Car Search Functionality
🎯 Advanced Filters for Cars
📈 Data Visualization with Plotly
🚘 Car Price & Mileage Analysis
📂 Dataset Explorer
⬇️ Download Filtered Data as CSV
📉 Fuel, Transmission & Owner Insights
🛠️ Technologies Used
Python
Streamlit
Pandas
Plotly Express
Streamlit Option Menu

🚀 Installation

Clone the repository:

git clone https://github.com/your-username/cardekho-dashboard.git
cd cardekho-dashboard

Install required libraries:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run app.py
